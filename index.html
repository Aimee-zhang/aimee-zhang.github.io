javascript 变量

什么是变量呢？

所谓变量，就是其值是可以改变的，变量可以看做是存储数据的容器。比如一个瓶子，它既可以装入酱油、醋；也可以装入茅台和二锅头……

如何定义变量

几种常用的数据类型：（前三种常用😄）

number 值是数值；

string 值是字符值；

boolean值为布尔值（true，false）;

undefined 值未定义；

null 值为空；

变量的命名规范

变量命名规范：

1.区分大小写，Name和name是两个不同的标识符。

2.不能数字开头，必须以字母、下划线或者美元符号开始，不能使用特殊符号。

3.命名不能是系统的关键字

4.中间不能有空格

有些保留字也不可以用来命名：如name

关键字可用于表示控制语句的开始或结束，或者用于执行特定操作等。
按照规则，关键字也是语言保留的，不能用作标识符

onclick事件的绑定

我们在浏览网页时,经常会通过点击鼠标来进行一些操作,比如点击登录按钮进行账 号的登录,点击注册按钮进入到一个账号注册页面,我们如何来实现一个点击时间 呢?这里我们就需要用到onclick事件绑定。
示例:点击红色div,弹出一个对话框

onmouseover、onmouseout事件

鼠标基本事件除了onclick之外,还有:onmouseover、onmouseout事件。

onmouseover表示鼠标移入,在鼠标移入元素时会触发该方法;

onmouseout表示鼠标移出,在鼠标移出元素时,会触发该方法。
